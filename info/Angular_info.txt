Установка Angular:
1) Установить Node.js
2) Проверить в командной строке node -v, npm -v. Если версия прописывается, то всё установлено правильно.
   Если версия не прописывается и выходит ошибка “command not found” - нужно добавить пути в переменную path 
   Поиск - Система - Доп.параметры - Переменные среды
   Для npm: c:\Program Files\nodejs
   Для ng: c:\Users\[ваш пользователь]\AppData\Roaming\npm
3) Далее установить Angular командой npm install -g @angular/cli (источник: https://angular.io/guide/setup-local)
   В результате установится ng, с помощью которой мы будем создавать компоненты, запускать сервер, обновлять
4) В командную строку вводим команду ng для проверки или ng --version (значит Angular CLI установлен)
   Если команды не работают, снова проверить переменные среды. В path верхней части должна появиться запись - 
   C:\Users\Vladimir\AppData\Roaming\npm
   
Создание проекта Angular:
1) cd C:\Users\Vladimir\IdeaProjects

3) cd angular-basics

5) Доступ к проекту http://localhost:4200/

Структура Angular-проекта:
node-modules - файлы фреймворка
src - исходный код, дополнительные файлы проекта
angular.json - настройки приложения
polyfills.ts  - настройки браузеров
main.ts - настройки по запуску
assets - папка для файлов (изображения, стили, всё что не входит в стандартный набор модулей)
app - компоненты
Компонент - основной элемент 
1) Класс
2) Html
3) Настройки компонента

@Component - декоратор, добавляет к классу метаданные. Их распознаёт движок и понимает, что это класс визуального компонента
